About this tool
Drug information is sourced from the FDA drug label database (OpenFDA) — no account or API key required. Data reflects US prescribing information; always verify against current Australian guidelines, AMH, or MIMS before clinical use.
Data source
  1. Search uses api.fda.gov/drug/label.json
  2. Australian drug names are automatically mapped where they differ (e.g. noradrenaline → norepinephrine, paracetamol → acetaminophen)
  3. If a drug isn't found, try the generic name or alternate spelling
  4. Label text may be verbose — key sections are trimmed for readability
