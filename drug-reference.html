Anthropic API Key
Drug information is generated by Claude (claude-haiku-4-5) — Australian clinical context, metric doses, Australian drug names. Your API key is stored in your browser only and never sent anywhere except Anthropic's API.
Getting an API key
  1. Go to console.anthropic.com
  2. Sign in or create a free account
  3. Navigate to API Keys and create a new key
  4. Add credits under Billing — Haiku is very cheap (~$0.001 per lookup)
  5. Paste the key above and click Save
